# 🐱 POMODOLLDOLL

고양이·강아지와 함께하는 코지·세피아톤 뽀모도로 & 태스크 데스크탑 앱 (Electron).
(이전 이름: CozyPomodoro → 4myo1dolldoll)

## 기능
- 뽀모도로 30분 ~ 2시간 슬라이더 조절, 화면을 옮겨도 흐르는 종료시각 기반 타이머
- **장면 6종 선택** (우측 상단 `🎨 장면` 버튼)
  - 기본 고양이 / 치즈 고양이 / 회색 고양이 / 턱시도 고양이 → 야옹 소리
  - 비숑 강아지 → 멍멍 소리
  - 뽀모도로 시계 UI → 알람 소리
  - 모든 효과음은 합성음이라 음원 파일이 필요 없습니다
- **프로젝트 + 하위 태스크** 관리
  - 프로젝트 최대 4개, 프로젝트마다 하위 태스크 최대 5개
  - `＋` 버튼으로 하나씩 추가, 작성하지 않은 칸은 보이지 않음
  - 각 항목의 `✕`로 삭제
- **집중 대상 직접 선택**: 태스크 왼쪽의 동그라미(◉)로 어떤 일에 집중할지 고르면, 그 일이 "지금 집중하는 일"로 표시됩니다 (미선택 시 첫 번째 항목)
- 완료한 태스크는 리스트에서 사라지고 그날 완료 기록으로 저장
- **음악 탭**: 앱 안에서 YouTube Music / Spotify / SoundCloud 사용 (로그인 유지)
- 깃허브 잔디 스타일 달력 (세피아 5단계 농도), 날짜별 세션 수(🍅) 표시
- 날짜 클릭 시 그날 완료한 태스크 상세 보기 + **기록 삭제** (수정은 불가, 삭제만)
- 완료된 세션만 기록, 중도 정지는 기록하지 않음
- 프레임리스 창 + 시스템 트레이 상주 + 항상 위에 표시 토글

데이터는 앱 `localStorage`에 저장됩니다. (기존 CozyPomodoro 데이터는 처음 실행 시 자동 이전됩니다.)

### 음악 탭 참고
- 내 YouTube Music 계정으로 로그인하면 세션이 유지됩니다.
- ⚠ **Spotify**는 보호된 스트리밍(Widevine DRM)을 사용해 표준 Electron 환경에서는 재생이 제한될 수 있습니다. 그럴 땐 YouTube Music / SoundCloud를 이용하세요.

## 개발 실행
```bash
npm install
npm run icon     # build/icon.svg -> icon.png / icon.ico 생성
npm start
```

## 빌드 (Windows)
```bash
npm run build:win        # NSIS 설치본 + 포터블 둘 다
npm run build:portable   # 포터블 단일 exe만
```
결과물은 `dist/` 폴더에 생성됩니다:
- `POMODOLLDOLL-1.1.0-x64.exe` (설치 프로그램)
- `POMODOLLDOLL-portable-1.1.0.exe` (설치 없이 실행)

> `appId`를 그대로 유지해, 기존 **CozyPomodoro 1.0.0** 설치본 위에 그대로 업그레이드 설치됩니다.

## 업데이트 배포 (수동)
자동 업데이트는 사용하지 않고, 새 버전은 직접 나눠줍니다.

1. `package.json`의 `version`을 올립니다 (예: `1.1.0` → `1.1.1`).
2. `npm run build:win` 으로 설치본을 만듭니다.
3. `dist/`의 새 `.exe`를 받는 사람에게 전달합니다.
4. 받은 사람이 실행하면, `appId`가 같아 **기존 설치 위에 덮어쓰기 업그레이드**됩니다. (`localStorage` 데이터는 유지)

## 사용
- 창 상단 바를 드래그해 이동, 우측 버튼으로 📌 항상 위 / 최소화 / 트레이로 숨기기
- 트레이 아이콘 클릭으로 보이기·숨기기, 우클릭 메뉴에서 종료
