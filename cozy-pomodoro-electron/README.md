# 🐱 CozyPomodoro

고양이와 함께하는 코지·세피아톤 뽀모도로 & 태스크 데스크탑 앱 (Electron).

## 기능
- 뽀모도로 30분 ~ 2시간 슬라이더 조절
- 집중 종료 시 합성된 고양이 울음소리 (음원 파일 불필요)
- 오늘의 태스크 5개 체크박스 + 집중 시 현재 태스크 외 흐리게
- 완료한 태스크는 리스트에서 삭제 → 그날 완료 기록으로 저장
- 깃허브 잔디 스타일 달력 (세피아 5단계 농도), 날짜별 세션 수(🍅) 표시
- 날짜 클릭 시 그날 완료한 태스크 상세 보기
- 뽀모도로 / 달력 / 기록 3화면 토글
- 화면을 옮겨도 세션은 계속 흐름 (종료시각 기반 타이머)
- 완료된 세션만 기록, 중도 정지는 기록하지 않음
- 프레임리스 창 + 시스템 트레이 상주 + 항상 위에 표시 토글

데이터는 앱 `localStorage`에 저장됩니다.

## 개발 실행
```bash
npm install
npm run icon     # build/icon.svg -> icon.png / icon.ico 생성
npm start
```

## 빌드 (Windows)
```bash
npm run build:win        # NSIS 설치본 + 포터블 둘 다
npm run build:portable   # 포터블 단일 exe만
```
결과물은 `dist/` 폴더에 생성됩니다:
- `CozyPomodoro Setup 1.0.0.exe` (설치 프로그램)
- `CozyPomodoro-portable-1.0.0.exe` (설치 없이 실행)

## 사용
- 창 상단 바를 드래그해 이동, 우측 버튼으로 📌 항상 위 / 최소화 / 트레이로 숨기기
- 트레이 아이콘 클릭으로 보이기·숨기기, 우클릭 메뉴에서 종료
