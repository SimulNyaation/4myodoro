# 🐱 POMODOLLDOLL

고양이·강아지와 함께하는 코지·세피아톤 뽀모도로 & 태스크 데스크탑 앱 (Electron).

---

## 기능

- **뽀모도로 타이머** — 30분 ~ 2시간 슬라이더로 집중 시간 조절. 창을 옮기거나 백그라운드로 보내도 정확히 흐르는 종료시각 기반 타이머.
- **장면(타이머 스킨) 3종** — 우측 상단 아이콘 버튼으로 전환
  - 🐱 **CAT** — 가운데 고양이, 남은 시간은 원형 게이지로만 표현 / 완료 시 야옹 소리
  - 🐶 **DOG** — 가운데 강아지, 원형 게이지 / 완료 시 멍멍 소리
  - ⏱ **POMO** — 가운데 디지털 시간 + 원형 게이지 / 완료 시 알람 소리
  - 모든 효과음은 합성음이라 별도 음원 파일이 필요 없습니다.
- **프로젝트 + 하위 태스크 관리** — 프로젝트 최대 4개, 프로젝트마다 하위 태스크 최대 5개. `＋`로 추가하고 `✕`로 삭제, 빈 칸은 보이지 않습니다.
- **집중 대상 선택** — 태스크를 클릭하면 진하게 강조되어 그 일에 집중합니다. 집중을 시작하면 다른 항목은 숨겨지고 선택한 일만 보입니다.
- **음악 탭** — 앱 안에서 YouTube Music을 바로 사용합니다. 처음 한 번 로그인하면 계정이 유지됩니다.
- **달력 기록** — 깃허브 잔디 스타일 달력(세피아 5단계). 날짜별 완료 세션 수(🍅)가 표시되고, 완료된 세션만 기록됩니다(중도 정지는 기록 안 됨).
- **기록 삭제** — 날짜를 누르면 그날 완료한 태스크 상세를 볼 수 있고, 우측 상단 🗑 버튼으로 여러 개를 선택해 삭제할 수 있습니다.
- **위젯형 창** — 프레임리스 창, 시스템 트레이 상주, 항상 위에 표시 토글.

데이터는 앱 `localStorage`에 저장됩니다.

## 설치 방법

### 1. 이미 만들어진 설치본으로 설치 (권장)

`dist/` 폴더(또는 전달받은 파일)의 실행 파일을 사용합니다.

- `POMODOLLDOLL-1.1.0-x64.exe` — 설치 프로그램 (설치 후 시작 메뉴·바탕화면 바로가기 생성)
- `POMODOLLDOLL-portable-1.1.0.exe` — 설치 없이 바로 실행되는 단일 실행 파일

설치 프로그램을 실행하고 안내를 따르면 끝입니다.

### 2. 소스에서 직접 실행 / 빌드

[Node.js](https://nodejs.org) 18 이상이 필요합니다.

```bash
# 저장소를 받은 뒤 앱 폴더로 이동
cd cozy-pomodoro-electron

# 의존성 설치
npm install

# 개발 모드로 바로 실행
npm start

# Windows 설치본 + 포터블 빌드 (결과물은 dist/ 에 생성)
npm run build:win
```

## 사용법

1. **집중 시간 설정** — 슬라이더로 30분 ~ 2시간 중 원하는 시간을 정합니다.
2. **할 일 정리** — `＋ 프로젝트 추가`로 프로젝트를, `＋ 할 일 추가`로 하위 태스크를 만듭니다.
3. **집중 대상 선택** — 집중할 태스크를 클릭하면 진하게 강조됩니다.
4. **집중 시작** — `집중 시작`을 누르면 타이머가 흐르고, 선택한 일만 화면에 남습니다. 일시정지·정지도 가능합니다.
5. **완료 처리** — 태스크 왼쪽 체크박스를 누르면 완료 처리되어 그날의 기록으로 저장됩니다.
6. **음악** — `음악` 탭에서 YouTube Music을 켜고 들으며 집중하세요.
7. **기록 보기·삭제** — `달력` 탭에서 날짜를 누르면 그날의 기록을 보고, 🗑 버튼으로 필요 없는 기록을 골라 삭제할 수 있습니다.

### 창 조작

- 상단 바를 드래그해 창을 옮깁니다.
- 우측 상단 버튼: 📌 항상 위에 표시 / ─ 최소화 / ✕ 트레이로 숨기기
- 트레이 아이콘을 클릭하면 보이기·숨기기, 우클릭 메뉴에서 종료할 수 있습니다.
